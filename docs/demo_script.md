# Semora Demo Script

This document guides developers through running a live demonstration of Semora.

---

## Steps:
1.  **Initialize Semora**:
    *   Run `semora init` in a repository.
2.  **Make Code Modification**:
    *   Introduce changes to `sample_target_repo/main.py`.
3.  **Attempt Git Commit**:
    *   Stage the files and commit.
4.  **Observe Quality Gate**:
    *   See specs generate, run, and security evaluations output.
5.  **Review Dashboard**:
    *   Inspect synchronized runs and matrix updates on the dashboard.
